# Sentra App - Frontend Assignment

Angular application with authentication and user dashboard.

## Features

- Login page with form validation
- Protected dashboard route
- User list with pagination
- Token-based authentication with expiry handling
- Loading states and error handling with retry option
- HTTP interceptors for auth token and error handling
- Logout functionality

## Tech Stack

- Angular 21
- TypeScript
- Angular Signals (state management)
- RxJS

## API

- JSONPlaceholder API: https://jsonplaceholder.typicode.com/users

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm start
```

Open browser at `http://localhost:4200`
